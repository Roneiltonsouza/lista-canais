# 📺 VidYou IPTV - Lista Canais Ao vivo

Este repositório armazena e gerencia a lista de reprodução IPTV oficial para o projeto **VidYou**.

## 🚀 Como Adicionar Canais
Para adicionar novos conteúdos, edite o arquivo `lista.m3u` seguindo o padrão abaixo:

#EXTINF:-1 tvg-id="ID" tvg-name="NOME",NOME DO CANAL
